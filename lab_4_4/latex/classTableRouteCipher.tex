\doxysubsection{Подробное описание}
Класс для шифрования табличным маршрутным методом 

Шифрование осуществляется путём записи текста в таблицу по строкам и чтения по столбцам. \begin{DoxyWarning}{Предупреждения}
Реализация только для английского языка 
\end{DoxyWarning}
